## About The Project

GetVaccine is a Mobile App that help users to book vaccination and built using Flutter.

## Features

- [x] Login & Register
- [x] Vaccination News
- [x] Book Vaccination
- [x] Add Family Member
- [x] Vaccine Pass
- [x] Search Near Facility
